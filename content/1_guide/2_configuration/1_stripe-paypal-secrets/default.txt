## Set Stripe/PayPal secrets

To use Stripe and/or PayPal you need to set the respective keys. Read the [Stripe](https://docs.stripe.com/keys) and [PayPal](https://developer.paypal.com/api/rest/#link-getclientidandclientsecret) documentation on how to get the keys.


<code caption="/site/config/env.php">
```php
<?php

return [
	'wagnerwagner.merx.stripe.test.publishable_key' => 'pk_test_xxx…',
	'wagnerwagner.merx.stripe.test.secret_key' => 'sk_test_xxx…',
	'wagnerwagner.merx.stripe.live.publishable_key' => 'pk_live_xxx…',
	'wagnerwagner.merx.stripe.live.secret_key' => 'sk_live_xxx…',
	'wagnerwagner.merx.paypal.sandbox.clientID' => 'xxx…',
	'wagnerwagner.merx.paypal.sandbox.secret' => 'xxx…',
	'wagnerwagner.merx.paypal.live.clientID' => 'xxx…',
	'wagnerwagner.merx.paypal.live.secret' => 'xxx…',
];
```
</code>


We recommend to set these keys in `/site/config/env.php` on your server only. For your development environment you can use a separate env file containing only the sandbox keys.

<callout theme="caution">
Anyone can use your live mode secret API keys to make any API call on behalf of your account, such as creating a charge or performing a refund. Keep your keys safe.
</callout>
