## Namechanges

### Merx methods

`Merx::initializePayment` → [`Merx::initializeOrder`](reference/classes/merx/initialize-order)
`Merx::completePayment ` → [`Merx::createOrder`](/reference/classes/merx/create-order)

### hooks

`initializePayment:before` → [`initializeOrder:before`](/reference/options/hooks/initialize-order-before)
`initializePayment:after` → [`initializeOrder:after`](/reference/options/hooks/initialize-order-after)
`completePayment:before` → [`createOrder:before`](/reference/options/hooks/create-order-before)
`completePayment:after` → [`createOrder:after`](/reference/options/hooks/create-order-after)

Apart from that, there are many more [new hooks](/reference/options/hooks).

### formatPrice()

`formatPrice` or `Merx::formatPrice` → [`Merx::formatCurrency`](/reference/classes/merx/format-currency)

You can create your own helper function.

<code caption="/site/config/config.php">
```php
<?php
if (!function_exists('formatPrice')) {
  function formatPrice(?float $price) {
    return Merx::formatCurrency($price, 'EUR');
  }
}
```
</code>

### $cart->getTaxRates()

`$cart->getTaxRates()` → [`$cart->taxRates()`](/reference/classes/list-items/tax-rates)

### $orderPage->invoiceDate()

`$orderPage->invoiceDate()` → [`$orderPage->created()`]

## Payment methods

Payment method `credit-card-sca` is now `credit-card`.

## Merx messages

`Merx::setMessage()`and `Merx::getMessage()` are no longer available.
