## Create new page

Merx comes with a [product blueprint](reference/blueprints/pages/product). Create a new page in the panel and select *Product* as template. Set a price for your product.

## Template

By default the page model of product templates is [`Wagnerwagner\Merx\ProductPage`](/reference/classes/product-page). If you want to, you can create your own page model and extend the default [`Wagnerwagner\Merx\ProductPage`](/reference/classes/product-page) model.

You are completely free to build your very own template that fits to your product. Most likely you want to display the price of the product and a button to add the product to the cart. You can use [`$page->price()`](/reference/classes/product-page/price) to display the formatted price, with currency symbol.

<code caption="/site/templates/product.php">
```php
<?php
/** @var Wagnerwagner\Merx\ProductPage $page */
?>
…
<span class="price"><?= $page->price() ?></span>
<form id="product-form" action="<?= url('api/shop/cart') ?>" method="post">
  <input type="hidden" name="page" value="<?= $page->uuid() ?>">
  <input type="number" name="quantity" min="1" value="1">
  <button>Add to cart</button>
</form>
…
```
</code>

## Add the product to the cart

Submitting the product form will create a POST request to the [Merx API endpoint](/reference/api/routes/shop-cart-post) and responds as JSON. Use JavaScript to interfere the submit event and log the response data to the console for now.

<code caption="script.js">
```js
const productForm = document.getElementById('product-form');
productForm.addEventListener('submit', async (event) => {
  event.preventDefault();
  const response = await fetch(productForm.action, {
    method: 'post',
    body: new FormData(productForm),
  });
  const json = await response.json();
  console.log(json.data);
});
```
</code>

Continue with [Showing the cart](/guide/getting-started/showing-the-cart) and learn how to use JavaScript to show the cart to the user.

If you don’t want to use JavaScript continue with [Using the API with PHP](/guide/api/using-api-with-php).
